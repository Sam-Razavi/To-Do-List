# To-Do-List

## Features

-   Add tasks with due dates
-   Mark tasks as completed
-   Delete tasks
-   Filter tasks by status (completed, pending)
-   Sort tasks by due date

## Installation

    ```sh
    git clone https://github.com/Sam-Razavi/To-Do-List
    ```
